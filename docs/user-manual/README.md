## 🌟 核心特性

### 免费无限存储
- **无存储限制** - 基于 Telegram 的存储机制，理论上无限容量
- **无需付费** - 在 Cloudflare 免费额度内完全免费使用
- **全球访问** - 利用 Cloudflare CDN 实现全球快速访问

### 多格式支持
- **图片格式** - JPG, PNG, GIF, WebP, BMP 等
- **文档格式** - PDF, DOC, TXT 等（作为文档上传）
- **媒体格式** - MP4, MP3, AVI 等（作为媒体文件上传）
- **大小限制** - 单文件最大 5MB（Telegram 限制）

### 智能管理
- **三种视图** - 列表视图、网格视图、瀑布流视图
- **批量操作** - 支持批量删除、复制链接等操作
- **搜索功能** - 按文件名、上传时间等条件搜索
- **状态管理** - 黑白名单、审查状态等

## 🔧 使用前准备

### 访问地址
确保您知道正确的访问地址：
- **主站地址** - 您的 Telegraph-Image 部署域名
- **后台地址** - `您的域名/admin`（需要管理员权限）

### 浏览器要求
推荐使用以下浏览器以获得最佳体验：
- **Chrome** 90+ （推荐）
- **Firefox** 88+
- **Safari** 14+
- **Edge** 90+

### 网络要求
- **稳定网络连接** - 确保网络连接稳定
- **合理网速** - 上传大文件时需要足够的上传带宽
- **防火墙设置** - 确保可以访问 Cloudflare 和 Telegram 服务

## 📱 移动端使用

Telegraph-Image 支持移动端访问：

### 移动浏览器
- 支持主流移动浏览器
- 响应式设计，自适应屏幕尺寸
- 触摸友好的操作界面

### 功能支持

## 🔒 隐私和安全

### 数据安全
- **传输加密** - 所有数据传输使用 HTTPS 加密
- **存储安全** - 文件存储在 Telegram 服务器，享受其安全保护
- **访问控制** - 支持黑白名单和权限管理

### 隐私保护
- **匿名上传** - 默认情况下不记录用户身份信息
- **可选遥测** - 可通过环境变量禁用数据收集
- **内容审查** - 可选的内容审查功能保护用户和访问者

## 📊 使用限制

### Cloudflare 限制
- **请求数量** - 免费版每日 100,000 次请求
- **带宽限制** - 根据 Cloudflare 政策
- **存储限制** - KV 存储有读写次数限制

### Telegram 限制
- **文件大小** - 单文件最大 5MB
- **API 限制** - 受 Telegram Bot API 速率限制
- **格式限制** - 某些特殊格式可能不支持

### 使用建议
- **合理使用** - 避免滥用服务
- **监控用量** - 定期检查使用量统计
- **备份重要文件** - 重要文件建议多重备份
