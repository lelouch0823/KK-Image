## 🚀 快速开始

### 首次登录

1. **访问管理界面**
   ```
   https://your-domain.pages.dev/admin
   ```

2. **输入认证信息**
   - 用户名：环境变量 `BASIC_USER` 的值
   - 密码：环境变量 `BASIC_PASS` 的值

3. **验证功能**
   - 确认可以看到图片列表
   - 测试基本操作功能
   - 检查权限设置

### 基础配置检查

**环境变量验证**:
```bash
# 必需的管理员环境变量
BASIC_USER=admin
BASIC_PASS=your-secure-password

# 可选的功能配置
ModerateContentApiKey=your-api-key
WhiteList_Mode=false
disable_telemetry=false
```

**KV 存储检查**:
- 确认 KV 命名空间已创建
- 验证绑定配置正确
- 测试读写权限

**Telegram 配置验证**:
- 确认 Bot Token 有效
- 验证频道权限
- 测试文件上传功能

## 🆘 故障处理

### 常见问题

**上传失败**:
- Telegram API 连接问题
- 文件大小或格式限制
- 网络连接异常
- 权限配置错误

**访问异常**:
- CDN 缓存问题
- DNS 解析异常
- SSL 证书问题
- 服务器负载过高
